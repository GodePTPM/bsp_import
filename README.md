## BSP files importer for Blender

# How to use   

## For BSPv30 files

Place the .bsp file to the root directory with .wad files

 **Structure:**
```/file.bsp```
```/file.wad```

## For RBSP files

Unpack the .pk3 archive and place the .bsp file to the root directory

 **Structure:**
 ```/file.bsp```
```/textures/some/file.png```
```/models/some/file.png```
